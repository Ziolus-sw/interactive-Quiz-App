# 🧠 Interactive Quiz App

A fully functional quiz application built using HTML, CSS, and Vanilla JavaScript. This project was created to strengthen my JavaScript skills by implementing state management, dynamic rendering, event delegation, and interactive user flows without relying on external frameworks.

---

## ✨ Features

- Multiple quiz screens
  - Start Screen
  - Quiz Screen
  - Result Screen

- Dynamic question rendering
- Progress tracking
- Score calculation
- Previous/Next navigation
- Answer persistence
- Correct and incorrect answer highlighting
- Restart functionality
- Responsive design
- Personalized result messages

## 🎯 Project Goal

The goal of this project was to move beyond basic JavaScript exercises and build a complete interactive application from scratch.

Rather than focusing only on functionality, this project emphasized:

- Managing application state
- Keeping the UI synchronized with data
- Creating a smooth user experience
- Building reusable rendering logic

---

## 🚀 Features Implemented

### Quiz Engine

- Questions generated dynamically from JavaScript data
- Score updates in real time
- Navigation between questions
- Prevents changing answers after submission
- Stores user selections

### User Experience

- Progress indicator
- Instant answer feedback
- Correct answer highlighting
- Encouraging result messages
- Restart and replay support
